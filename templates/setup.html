Music Recognition Setup

This application uses AcoustID for audio fingerprinting and music recognition. You need to provide an API key from AcoustID to enable this feature.

To get an API key:

  1. Go to acoustid.org/login and create an account
  2. After logging in, go to acoustid.org/applications
  3. Register a new application, providing a name (e.g., "Pi Music Display") and a version
  4. Copy the API key and paste it below
